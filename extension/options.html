Flow2API Captcha Worker

为当前 Chrome 实例设置独立路由。扩展可以给 Flow2API 提供验证码，也可以把当前已登录的 Google Flow 账号导入账号池，并同步 Google Cookies 用于协议自动刷新。

打码模式使用 WebSocket URL 和 API Key。 账号导入使用 Account Import URL 和 Account Import Token。
导入账号前，请先在任意 Chrome 标签页打开已登录的 Google Flow 项目页面，并确认同一 Chrome 已登录 accounts.google.com。扩展会自动读取必要的 Google Cookies；如果没读到，会按仅 ST 模式导入。
